## 如何部署？

### 1. clone 项目

```bash
git clone https://github.com/Skillw/Skillw.git
```

### 2. 禁书目录，启动!

```bash
cd Skillw
npm install
npm run dev
```

当然，你可以无缝切换到pnpm, yarn, 甚至是bun.
