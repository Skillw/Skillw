      • getSnapshotInventory

        @NotNull() abstract BrewerInventory getSnapshotInventory()

        获取本容器的物品栏的快照.

        返回的快照不与任何方块相关联. 任何对此物品栏快照的修改将不会应用于真实的方块上, 直到调用 update.

        原文:Gets the captured inventory snapshot of this container.

        The returned inventory is not linked to any block. Any modifications to the returned inventory will not be applied to the block represented by this block state up until update has been called.
