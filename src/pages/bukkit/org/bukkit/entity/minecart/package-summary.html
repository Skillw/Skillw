Package org.bukkit.entity.minecart

为不同种类的矿车提供的接口.

See: Description

  • Interface Summary 
    Interface Description
    CommandMinecart代表命令方块矿车.
    ExplosiveMinecart代表TNT矿车，可以被引爆.
    HopperMinecart代表漏斗矿车.
    PoweredMinecart代表动力矿车.
    RideableMinecart代表可乘坐的矿车.
    SpawnerMinecart代表有刷怪箱在里面的矿车（即刷怪箱矿车）.
    StorageMinecart代表运输矿车，你可以使用InventoryHolder里的方法来访问.
