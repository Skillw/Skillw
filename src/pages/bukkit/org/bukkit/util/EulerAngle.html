Class EulerAngle

  • All Implemented Interfaces:
    
public class EulerAngle

    EulerAngle即欧拉角,是指用给定的三个角, 每一个表示一个轴(x,y,z)的偏转角.这些角用弧度表示.

    原文:EulerAngle is used to represent 3 angles, one for each axis (x, y, z). The angles are in radians
