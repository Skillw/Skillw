Package org.bukkit.plugin.messaging

用于设定插件至客户端的消息传输协议的类.

See: Description

  • Class Summary 
    Class Description
    ChannelNameTooLongException如果插件通道(Plugin Channel)过长则抛出此异常。
    ChannelNotRegisteredException如果插件尝试向未注册的通道发消息则抛出此异常。
    MessageTooLargeException如果要发送的插件信息(Plugin Message)过大则抛出此异常。
    PluginMessageListenerRegistration包含了关于Plugin注册的插件通道的信息.
    ReservedChannelException如果插件尝试去注册一个预留通道则抛出此异常(例如"REGISTER通道")。
    StandardMessengerMessenger的标准实现.
  • Interface Summary 
    Interface Description
    Messenger负责管理注册插件通道和监听器的类.
    PluginMessageListener一个指定通道的一个监听器，将接收从客户端发送的消息的通知.
    PluginMessageRecipient代表插件消息(Plugin Message)可能的接收者.
  • Enum Summary 
    Enum Description
    PluginChannelDirection代表插件通道的不同指向。
