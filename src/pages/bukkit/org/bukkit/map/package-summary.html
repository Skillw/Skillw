Package org.bukkit.map

用于简化处理地图显示内容的类.

See: Description

  • Interface Summary 
    Interface Description
    MapCanvas代表一个画地图的画布，每个画布都关联着一个特定的 地图渲染器并表现为地图上渲染器的一层.
    MapColorCacheHolds cached information for matching map colors of a given RBG color.
    MapViewRepresents a map item.
  • Enum Summary 
    Enum Description
    Type表示地图光标的标准类型。可能有更多被资源包(原材质包)提供的类型，这个数值被客户端当做minecraft.jar里的'./misc/mapicons.png'文件的索引或一个材质包中的索引使用.
    ScaleAn enum representing all possible scales a map can be set to.
  • Class Summary 
    Class Description
    MapCursor表示一个地图上的光标.
    MapCursorCollectionRepresents all the map cursors on a MapCanvas.
    CharacterSpriteRepresents the graphics for a single character in a MapFont.
    MapFontRepresents a bitmap font drawable to a map.
    MapPaletteRepresents the palette that map items use.
    MapRendererRepresents a renderer for a map.
    MinecraftFontRepresents the built-in Minecraft font.
