Class Structure

    org.bukkit.Keyed 
    
public abstract class Structure
 implements Keyed

    Represent a Structure from the world. Listed structures are present in the default server. Depending on the server there might be additional structures present (for example structures added by data packs), which can be received via STRUCTURE.
