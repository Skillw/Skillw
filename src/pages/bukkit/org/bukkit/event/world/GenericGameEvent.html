public class GenericGameEvent
extends WorldEvent implements Cancellable

    代表 Mojang 通用游戏事件. 你应该尽可能地使用 Bukkit 框架提供的其它事件, 此事件在游戏内部主要用于Sculk_Sensor.

    原文:Represents a generic Mojang game event. Specific Bukkit events should be used where possible, this event is mainly used internally by Sculk sensors.
