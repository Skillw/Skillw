    IgniteCause枚举：方块被点燃的原因

  • Class Summary 
    Class Description
    BellResonateEventCalled when a bell resonated after being rung and highlights nearby raiders.
    BellRingEventCalled when a bell is being rung.
    BlockBreakEvent当一个方块被玩家破坏的时候，调用本事件.
    BlockBurnEvent当一个方块被火烧掉的时候触发此事件.
    BlockCanBuildEvent当我们尝试建造一个方块的时候，可以检测我们是否可以在此建造它。 注意:
    • getBlock()返回的方块是我们正试图放置在其上面的方块,不是我们试图放置的方块。
    • 如果你想知道玩家放置的是什么方块, 用 getMaterial 来代替.
    BlockCookEvent当物品堆在方块内成功烹饪时触发本事件 (原版中这个方块一般是营火).
    BlockDamageAbortEventCalled when a player stops damaging a Block.
    BlockDamageEvent方块被玩家损坏的事件.
    BlockDispenseArmorEvent当一个可装备物品从某个方块发射出来并装备到附近的实体上时触发本事件.
    BlockDispenseEvent物品被方块(比如发射器,投掷器等)射出的事件.
    BlockDropItemEvent当一个方块被玩家破坏且掉落物品时触发本事件.
    BlockEvent代表与方块相关的事件.
    BlockExpEvent方块产生经验的时候(比如煤矿被被打破会掉落经验)时触发.
    BlockExplodeEvent方块爆炸事件.
    BlockFadeEvent方块因为自然条件消退，融化，消失的事件.
    BlockFertilizeEvent当玩家使用骨粉向某方块施肥后导致方块发生变化时触发本事件.
    BlockFormEvent当一个方块因为自然变化被放置、更改或者蔓延时(比如下雪),触发此事件.
    BlockFromToEvent液体流动/龙蛋自己传送的事件(源方块到目标方块).
    BlockGrowEvent当一个方块在世界中自然生长的时触发此事件(如小麦生长).
    BlockIgniteEvent当一个方块被点燃时触发.
    BlockMultiPlaceEvent当玩家放一个方块, 而连锁放置了第二个方块时(例如玩家放置一个床)触发本事件.
    BlockPhysicsEvent方块物理事件(例如是沙子掉落、流水).
    BlockPistonEvent活塞相关事件
    BlockPistonExtendEvent活塞臂推出事件.
    BlockPistonRetractEvent活塞臂缩回事件
    BlockPlaceEvent当一个方块被玩家放置的时候触发此事件.
    BlockReceiveGameEvent当Sculk Sensor接收到游戏事件并因此可能激活时调用.
    BlockRedstoneEvent当方块接受到的红石信号变化时触发此事件
    BlockShearEntityEvent当发射器剪掉附近的羊的羊毛时触发本事件 (发射器内含剪刀时有机会触发本事件).
    BlockSpreadEvent当一个方块基于自然法则地蔓延时触发此事件(比如菌丝的蔓延).
    BrewingStartEventCalled when a brewing stand starts to brew.
    CampfireStartEventCalled when a Campfire starts to cook.
    CauldronLevelChangeEvent炼药锅水位发生变化时触发本事件.
    EntityBlockFormEvent实体形成方块时触发此事件 举个例子:
    • 雪傀儡移动时形成的雪
    • 穿戴冰霜行者附魔的鞋子形成的冰霜
    FluidLevelChangeEvent当液体方块的液面高度因相邻方块而发生变化时触发本事件.
    InventoryBlockStartEventUsed when:
    • A Furnace starts smelting FurnaceStartSmeltEvent
    • A Brewing-Stand starts brewing BrewingStartEvent
    • A Campfire starts cooking CampfireStartEvent
    LeavesDecayEvent当树叶消失时触发此事件.
    MoistureChangeEvent当耕地的湿润等级发生变化时触发本事件.
    NotePlayEvent音符盒播放音符的事件[玩家点击/红石触发]
    SignChangeEvent在玩家设置牌子上的内容子时触发.
    SpongeAbsorbEvent当海绵吸水时触发本事件.
    TNTPrimeEventCalled when a block of TNT in the world become primed.
