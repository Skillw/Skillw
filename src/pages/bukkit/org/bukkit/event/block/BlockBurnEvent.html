Class BlockBurnEvent

  • All Implemented Interfaces:
    org.bukkit.event.Cancellable 
    
public class BlockBurnEvent
extends BlockEvent implements Cancellable

    当一个方块被火烧掉的时候触发此事件.

    若方块点燃事件被取消，此方块就不会被火烧毁.
