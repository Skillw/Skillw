Package org.bukkit.event.raid

与袭击有关的事件(Minecraft Wiki - 袭击).

See: Description

  • Class Summary 
    Class Description
    RaidEvent与袭击有关的事件.
    RaidFinishEvent当袭击以一个清晰的结果结束时触发本事件.
    RaidSpawnWaveEvent当新一波袭击开始时触发本事件.
    RaidStopEvent当袭击终止时触发本事件.
    RaidTriggerEvent当袭击被触发时触发本事件 (例如:带有不祥之兆效果的玩家进入村庄).
